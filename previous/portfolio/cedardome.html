Cedar Dome

cedardome.com

I worked on the Cedar Dome website in December 2013. The Cedar Dome is a very unique residence in Jacksonville, Illinois. It was built by a man named Buckminster Fuller in 1975, not long after he designed the religious center on the campus of SIUE.

My role on this project was only the development of the website. The biggest challenge for me was building the navigation so that the home icon would flex along with the rest of the links, so that it would be properly formatted at any screen size.
